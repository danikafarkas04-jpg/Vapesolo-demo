INCLUDED FILES:
- index.html (The main layout, styles, and scripts combined)
- README.txt (This file)

HOW TO USE & EDIT:
1. Open index.html in your favorite code editor (e.g., VS Code).
2. All CSS is located in the <style> tag at the top.
3. All JavaScript (including the 3D interactions and product data) is located in the <script> tag at the bottom.

EDITING PRODUCTS:
To change the products, scroll down to the JavaScript section. Look for the "const products" array. 
You can easily change the names, subheadings, and colors (Hex codes) for each product there. The 3D models will automatically update based on your color inputs!

COLORS:
- c1, c2: The main gradient of the front face.
- side: The darker side color.
- nozzle: The top mouthpiece color.
- glow, led, ac: Accent colors for shadows and buttons.
